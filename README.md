### ✨ Requirements
* **OS:** Android 10 or higher

### 🛠 Tech Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose
- **Architecture:** Clean Architecture + MVVM
- **Concurrency:** Kotlin Coroutines, Flow, Channels
- **Dependency Injection:** Dagger Hilt
- **Storage** Preferences DataStore
